## Overview

Format your documentation content using standard Markdown syntax and custom MDX components.

### Basic Text Styles

Apply formatting using standard Markdown syntax:

- **Bold text**: `**bold text**`
- *Italic text*: `*italic text*`
- `Inline code`: `` `inline code` ``
- ~~Strikethrough~~: `~~strikethrough~~`
